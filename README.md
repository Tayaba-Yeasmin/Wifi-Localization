#  WiFi Indoor Localization using ML vs Deep Learning

## Project Overview
This project performs indoor localization using WiFi signal strength (RSSI values). It compares traditional Machine Learning (Random Forest) with Deep Learning (Neural Network) to predict room/floor locations.

---

## Dataset
UCI Wireless Indoor Localization Dataset  
Each sample contains WiFi signal strengths from multiple access points.

---

## Objectives
- Predict indoor location using WiFi signals
- Compare ML and DL performance
- Analyze accuracy and learning behavior

---

### Machine Learning
- Random Forest Classifier

### Deep Learning
- Fully Connected Neural Network (Dense layers)
---

## Visualizations
- Accuracy comparison bar chart
- Training curve for Deep Learning

---
## Key Learnings
- WiFi signals can be used for indoor positioning
- ML works well for structured signal data
- DL captures hidden nonlinear patterns
